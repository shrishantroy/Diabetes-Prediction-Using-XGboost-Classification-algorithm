# Diabetes-Prediction-Using-XGboost-Classification-algorithm
A Machine Learning model that predicts whether a patient has diabetes or not. The model is trained using a dataset of 8 features and 768 samples. 80 percent of which is used to train the model and 20 percent used of test the model for its performance. The model is found to be 75 percent accurate with a good f1 score of 0.73. Its 78% of healthy predictions are correct while 70% of diabetic predictions are correct. It predicts 83% of healthy patients as healthy while it predicts only 63% of the diabetic patients as diabetic, which is the main drawback here. It misses 37% of diabetic patients and predicts them as healthy. Future developments can be done like Feature scaling and Feature engineering to improve the performance.
